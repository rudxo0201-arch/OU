## 콘텐츠 제작 가이드라인

### 숏폼 영상 포맷

```
[0-2초]  후킹: "가계부 앱 지우세요" / "이거 실화?"
[2-12초] 시연: 화면 녹화 — 타이핑 → 뷰 생성 순간
[12-15초] CTA: "링크는 바이오에" + OU 로고
```

- 배경음: 미니멀 (lo-fi 또는 무음)
- 자막: 필수 (무음 재생 대비)
- 화질: 폰 녹화 OK (자연스러움 > 프로페셔널)

### 에브리타임 게시글 톤

```
❌ "OU는 AI 기반 개인 데이터 관리 플랫폼입니다"
✅ "시험 기간인데 정리가 안 돼서 이것저것 써봤는데
    이거 진짜 편하더라. 말하면 플래시카드가 자동으로 만들어짐
    링크 남겨둘게 ㅎㅎ"

→ 광고 느낌 0, 일상 후기 톤
→ ㅋㅋ ㅎㅎ 같은 구어체 사용
→ 스크린샷 1-2장 첨부
```

### 블로그 SEO 키워드

```
"시험 공부 방법" / "플래시카드 만들기" / "가계부 앱 추천"
"회의록 정리" / "습관 트래커" / "인맥 관리"
→ 각 키워드에 시나리오 체험 링크 삽입
```
