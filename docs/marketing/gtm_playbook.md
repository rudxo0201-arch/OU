## 1. GTM 전략

### 출시 단계

| 단계 | 시기 | 목표 | 핵심 지표 |
|------|------|------|-----------|
| **클로즈드 베타** | 2026년 5월 | 20명의 미친 팬 확보 | D7 리텐션 > 40% |
| **소프트 런치** | 2026년 6월 | 200명 DAU | 시나리오 완료율 > 60% |
| **풀 런치** | 2026년 8월 | 1,000명 DAU | Pro 전환율 > 10% |

### 초기 집중 채널: 2개만

1. **에브리타임 (대학생 커뮤니티)** — 타겟이 매일 있는 곳
2. **숏폼 영상 (릴스/쇼츠/틱톡)** — 15초 시나리오 데모

### 초기 고객 획득: "시험 기간 대학생"

```
왜 대학생인가:
- 시험 2주 전에 미친듯이 도구를 찾음
- 무료에 민감 → Free 티어로 진입
- 에브리타임에서 자연스럽게 공유
- 한 학과에서 1명만 쓰면 → 학과 전체로 전파
- 졸업 후 직장에서도 씀 → 장기 LTV

왜 시험 기간인가:
- "시험 공부"는 구체적 니즈 → 전환율 높음
- 플래시카드 자동 생성이 킬러 유스케이스
- 5월 중간고사 시즌에 맞춰 런치
```

### 런치 목표

```
클로즈드 베타 (5월):
  - 1개 대학 2개 학과, 각 10명 = 20명
  - 직접 리크루팅 (주변 지인, 에브리타임 DM)
  - D7 리텐션 40% 이상이면 소프트 런치 진행
  - 40% 미만이면 제품 수정 후 재시도

소프트 런치 (6월):
  - 5개 대학으로 확대
  - 에브리타임 게시 + 숏폼 영상 시작
  - 대학생 앰배서더 5명 모집

풀 런치 (8월):
  - 2학기 시작 시즌
  - 교수 1명 확보 → 수업 도구로 도입
  - Pro 플랜 오픈
```

## 2. 핵심 메시지

### 헤드라인

> **말하면, 정리돼요.**

### 서브헤드라인

> 일정, 가계부, 공부 노트, 할 일. 그냥 말하면 OU가 알아서 정리하고, 원하는 형태로 보여줘요.

### CTA

> **지금 체험하기** (가입 없이, 1초)

### 킬러 메시지 변형 (채널별)

| 상황 | 메시지 |
|------|--------|
| 대학생 시험 | "교재 내용 말하면 플래시카드가 자동으로" |
| 직장인 가계부 | "점심값 말하면 가계부 끝" |
| 직장인 회의 | "회의 끝나고 한마디면 회의록 완성" |
| 일반 | "캘린더? 가계부? 메모? 그냥 말하세요" |

### 메시지 검증 기준

```
✅ 타겟이 아닌 사람도 누구를 위한 건지 즉시 앎
✅ 3초 안에 "아 이런 건가" 이해됨
✅ "나도 해봐야겠다"는 행동 유발
❌ 기술 용어 (AI, 데이터, 온톨로지, 그래프) 절대 사용 금지
```

## 3. 채널별 전략

### 채널 A: 에브리타임

**선택 이유:** 대학생이 매일 접속. 학과별 게시판 → 타겟 정밀 조준 가능.

**콘텐츠 유형:**
- 후기형 게시글: "시험 공부에 이거 쓰니까 진짜 편함"
- 꿀팁형 게시글: "교재 내용 정리하는 새로운 방법"
- 질문형: "플래시카드 자동으로 만들어주는 거 써본 사람?"

**빈도:** 주 2회 (시험 기간엔 매일)

**성공 지표:**
- 게시글당 조회수 > 500
- 링크 클릭률 > 5%
- 가입 전환율 > 20% (클릭 대비)

**초기 3개월:**
- 5월: 직접 작성 (자연스러운 후기 톤)
- 6월: 앰배서더가 자기 학과 게시판에 작성
- 7월: 사용자 자발적 후기 유도 (Pro 1개월 무료 쿠폰)

### 채널 B: 숏폼 영상

**선택 이유:** "입력 → 결과"의 마법 순간이 15초 영상에 완벽히 맞음. 알고리즘 노출로 무료 도달 가능.

**콘텐츠 유형:**
- 시나리오 데모: "한마디로 캘린더 만들기" (15초)
- 비교형: "3분 vs 3초" (가계부 앱 입력 vs OU)
- 반응형: "친구한테 보여줬더니 반응" (UGC 유도)

**빈도:** 주 3회

**성공 지표:**
- 영상당 조회수 > 10K (첫 달 목표)
- 바이오 링크 클릭률 > 2%
- 댓글 "이거 뭐야?" 비율 > 10%

**초기 3개월:**
- 5월: 10개 시나리오 영상 제작 (직접)
- 6월: 사용자 반응 영상 수집, 리믹스
- 7월: 인플루언서 1명 시딩 (교육/생산성 크리에이터)

## 4. 바이럴 루프 설계

```
사용자 → 뷰 생성 → 공유 링크 → 받는 사람 체험 → 가입
  ↑                                              │
  └──────────────────────────────────────────────┘
```

### 공유 트리거 포인트

| 트리거 | 액션 | 예시 |
|--------|------|------|
| 뷰 생성 직후 | "이 뷰를 공유하시겠어요?" 토스트 | 캘린더 생성 → 공유 |
| 데이터 마일스톤 | "10개 기록 달성! 친구에게 자랑하기" | 습관 10일 연속 |
| 시나리오 완료 | "이 체험을 친구에게도" 버튼 | 가계부 시나리오 후 |

### 공유 링크 구조

```
ouuniverse.com/v/abc123  → 뷰 미리보기 (비로그인도 볼 수 있음)
                          → "나도 만들어보기" CTA
                          → 클릭하면 같은 시나리오로 체험 시작
```

## 5. 마케팅 역량 체계

### 필요 역량과 구축 방법

| 역량 | 왜 필요한가 | 구축 방법 | 우선순위 |
|------|------------|-----------|----------|
| **카피라이팅** | 10단어 안에 가치 전달 | 창업자가 직접 (제품을 가장 잘 아는 사람) | P0 |
| **숏폼 영상 제작** | 15초 시나리오 데모 | 화면 녹화 + 자막 (CapCut/VLLO) | P0 |
| **커뮤니티 운영** | 에브리타임/Reddit 침투 | 창업자 직접 + 앰배서더 5명 | P0 |
| **데이터 분석** | 퍼널 전환율 측정 | Vercel Analytics + Supabase 쿼리 | P1 |
| **SEO** | 검색 유입 (장기) | 블로그 콘텐츠 + 시나리오별 랜딩 | P2 |
| **파트너십** | 출판사/대학 B2B | Phase 3에서 별도 BD 역할 | P3 |

### Phase별 마케팅 조직

```
Phase 1 (MVP, ~200 DAU):
  - 마케팅 = 창업자 1인
  - 직접: 카피 + 숏폼 + 에브리타임
  - 도구: Canva, CapCut, Vercel Analytics

Phase 2 (~1,000 DAU):
  - + 대학생 앰배서더 5명 (Pro 무료 제공)
  - + 숏폼 크리에이터 1명 시딩
  - 도구: + 이벤트 트래킹 (Mixpanel/Amplitude)

Phase 3 (~10,000 DAU):
  - + 마케팅 담당 1명 채용 (콘텐츠 + 커뮤니티)
  - + BD 담당 1명 (출판사/대학 파트너십)
  - 앰배서더 → 20명 확대
```

### 최고 역량 구축 원칙

```
1. 창업자가 직접 한다 (초기)
   → 제품을 가장 잘 아는 사람이 메시지를 만든다
   → 외주/대행사는 제품을 모르기 때문에 메시지가 약해짐

2. 자동화할 수 있는 건 자동화한다
   → 시나리오 체험 → 뷰 생성 → 공유 유도: 전부 제품 안에서 자동
   → 마케팅의 70%가 제품 코드로 구현됨

3. 측정 → 반복 → 측정
   → 모든 마케팅 액션에 UTM 태그
   → 주간 리뷰: 어떤 메시지/채널에서 전환이 높았나
   → "감"이 아니라 "숫자"로 의사결정

4. 사용자의 말을 훔친다
   → 사용자가 OU를 설명하는 말 = 최고의 카피
   → "와 이거 그냥 말하면 되네" → 그대로 광고 카피로
```

## 6. 리스크와 대응

| 리스크 | 대응 |
|--------|------|
| 에브리타임 광고 신고 | 후기 톤 유지, 절대 광고 느낌 X |

| 앰배서더 이탈 | Pro 무료 + 추천인 수익 (Pro 전환 시 10% 커미션) |
| 대학생 방학 이탈 | 방학 전 습관/일기 시나리오로 일상 데이터 유도 |
| 경쟁사 카피 (Notion AI 등) | 속도. 먼저 사용자 기반 확보 + 데이터 락인 |
