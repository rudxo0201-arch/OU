## 시나리오 상세

### 시나리오 1: 직장인의 하루 (Just talk.)

```
유저: "내일 오후 3시에 강남역에서 미팅"
→ schedule 도메인, 일정 자동 생성

유저: "점심에 김치찌개 8000원"
→ finance 도메인, 지출 기록

유저: "요즘 좀 지치는 느낌"
→ emotion 도메인, 감정 기록

유저: 캘린더 뷰에서 내일 일정 확인
유저: 지출 차트에서 이번달 식비 확인
```

### 시나리오 2: 한의대생의 학습

```
유저: "마황의 성미귀경이 뭐야?"
→ LLM이 본초 DB에서 마황 정보 제공
→ 마황 DataNode 인라인 카드 표시

유저: "마황탕 조성 알려줘"
→ 방제 DB에서 마황탕 정보 + 구성 약재 연결

유저: 지식 그래프에서 마황 중심으로 관련 약재 탐색
유저: 플래시카드 뷰로 본초 암기
```

### 시나리오 3: 한자 학습

```
유저: "天地玄黃 宇宙洪荒"
→ 한자 8자 인라인 카드 자동 표시
→ 클릭하면 상세 (읽기, 부수, 자원 설명)

유저: "日月盈昃 辰宿列張"
→ 검색 모드 자동 전환 (직전에도 한자 검색했으므로)
→ LLM 스킵, 카드 뷰만 표시

유저: "天이 뭐야?"
→ 대화 모드 복귀, LLM 응답 + 天 인라인 카드
```

### 시나리오 4: 독서 기록

```
유저: "오늘 사피엔스 읽었는데 인지혁명 부분이 인상적이었어"
→ knowledge 도메인, 독서 노트 자동 생성
→ "사피엔스" 엔티티 노드 자동 생성

유저: "지난번에 총균쇠 읽을때도 비슷한 내용 있었는데"
→ 기존 "총균쇠" 노드와 "사피엔스" 노드 관계 연결
→ 지식 그래프에서 두 책 연결 확인
```
