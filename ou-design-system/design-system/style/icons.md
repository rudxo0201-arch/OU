# 아이콘

> KRDS 참조: 24px 기준, SVG, 선 굵기 스케일링

## 아이콘 라이브러리

**Phosphor Icons** (`@phosphor-icons/react`)
- 스타일: **Light** (기본) / **Fill** (활성 상태)
- 포맷: SVG (React 컴포넌트)

## 사이즈 체계

| 사이즈 | 값 | 용도 |
|--------|---|------|
| **XS** | 16px | 인라인 텍스트, 배지 안 |
| **SM** | 20px | 버튼 내 아이콘, 라벨 옆 |
| **MD** | 24px | **기본 — 네비게이션, 액션** |
| **LG** | 32px | 빈 상태, 강조 |
| **XL** | 48px | 히어로, 온보딩 |

> 모든 사이즈는 4px 또는 8px의 배수 **[DO]**

## 스타일 규칙 **[DO]**

```
기본 상태: Light (얇은 선)
활성 상태: Fill (채운 형태)
비활성 상태: Light + dimmed 색상

예시:
<House weight="light" />       ← 기본
<House weight="fill" />        ← 활성 (현재 탭)
<House weight="light" color="gray" /> ← 비활성
```

## 아이콘 + 텍스트 조합

| 조합 | 간격 | 정렬 |
|------|------|------|
| 아이콘 + 라벨 | 4~8px | 수직 중앙 |
| 버튼 내 아이콘 | 8px | 텍스트 왼쪽 |
| 네비게이션 | 아이콘만 (축소) / 아이콘+라벨 (확장) | — |

## 커스텀 아이콘 규칙

```
- 24px 그리드에서 제작
- 선 굵기: 1.5px (24px 기준)
- 모서리: brand 정체성에 맞게 (OU: 약간의 둥글기)
- SVG 파일로 관리
- viewBox: 0 0 24 24
```
